## Лабораторні (32 години)

Для лабораторних робіт робіть копії екранів для звітів. У тих пунктах, де ставляться запитання щодо їх виконання, записуйте відповіді у звіт. 

Звіти треба показувати і захищати викладачеві.

1. Конфігурування локальної та розподіленої системи I/O в Machine Expert.
   - [Встановлення середовища, створення та запуск проєкту EcoStruxure Machine Expert](https://github.com/asu-in-ua/atpv/blob/main/plc/ide/lab_mach_expert.md)

2. Програмування PLC в Machine Expert.
   - [Програмні сутності в Machine Expert і CODESYS](https://github.com/asu-in-ua/atpv/blob/main/plc/pou/labmachexpert.md)

3) Задачі керування та регулювання в PLC Machine Expert.

4) Мережний обмін в PLC Machine Expert. 

5) Робота з перетворювачами частоти.

6) Системи керування з функціями позиціонування.

7) Інтегрування з використанням OPC UA.

8) Інтегрування з використанням технологій IoT.
