# Програмування промислових контролерів

## Лекції (16 годин)

1. Апаратні засоби для контролю та керування машинами в системі промислового контролеру.
1. Основи програмування контролерів в середовищі Machine Expert.
1. Задачі регулювання в машинній автоматиці.
1. Промислові мережі в системах з PLC Machine Expert
1. Керування перетворювачами частоти з використанням промислових мереж.
1. Задачі керування позиціонуванням.
1. Розширені засоби програмування контролерів в середовищі Machine Expert.
1. Інтегрування в системи керування роботизованими системами.



## Лабораторні (32 години)

1. Конфігурування локальної та розподіленої системи I/O в Machine Expert.
2. Програмування PLC в Machine Expert.
3. Задачі регулювання в PLC Machine Expert.
4. Мережний обмін в PLC Machine Expert. 
5. Робота з перетворювачами частоти.
6. Системи керування з функціями позиціонування.
7. Інтегрування з використанням OPC UA.
8. Інтегрування з використанням технологій IoT.
